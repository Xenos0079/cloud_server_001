# Draw & Guess Game

一个轻量级的双人"你画我猜"网页游戏。

## 功能特性

- **双人模式**：第一个连接的玩家是"绘画者"，第二个是"猜测者"
- **异步流程**：绘画者本地绘制，完成后提交
- **绘画工具**：
  - 画笔：固定2px，黑色
  - 选择擦除器：点击拖拽创建矩形选择区域，虚线预览，释放后清除
- **游戏流程**：绘画者提交后，猜测者看到图片并输入答案

## 技术栈

- **后端**：Node.js + Express + Socket.io
- **前端**：Vanilla JavaScript + HTML5 Canvas + Tailwind CSS

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动服务器：
```bash
npm start
```

3. 在浏览器中访问：
   - 本地访问：`http://localhost:3000`
   - 局域网访问：`http://<服务器IP>:3000`

4. 游戏说明：
   - 第一个打开的浏览器窗口将成为"绘画者"
   - 第二个打开的浏览器窗口将成为"猜测者"
   - 绘画者需要先设置一个单词（用于测试），然后开始绘画
   - 绘画完成后点击"完成并提交"
   - 猜测者看到图片后输入答案并提交

## 注意事项

- 服务器监听在 `0.0.0.0:3000`，支持局域网访问
- 游戏状态存储在内存中，服务器重启后会重置
- 目前仅支持2人游戏，超过2人连接会收到"游戏已满"提示
